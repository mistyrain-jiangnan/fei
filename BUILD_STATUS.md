## 📱 安装到手机

构建完成后,您可以:

### 方式 1: 文件传输(推荐)

1. **找到 APK 文件**:
   - 脚本会自动打开文件夹
   - 或手动进入: `android\app\build\outputs\apk\debug\`

2. **发送到手机**:
   - 通过微信发给自己
   - 通过 QQ 文件助手
   - USB 复制到手机
   - 上传云盘后下载

3. **在手机上安装**:
   - 打开文件管理器
   - 找到 APK 文件
   - 点击安装
   - 允许安装未知来源(如有提示)

### 方式 2: ADB 安装

如果手机已开启 USB 调试并连接:

```powershell
.\install-apk.ps1
```
