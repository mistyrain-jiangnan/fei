# 快速开始 - 生成 APK 并安装到手机

## 最简单的方法 (3 个步骤)

### 步骤 1: 构建 APK

打开 PowerShell,运行:

```powershell
cd G:\fei
.\build-apk.ps1
```

**首次运行**需要 5-10 分钟(下载依赖)  
**后续运行**只需 1-3 分钟

### 步骤 2: 获取 APK 文件

构建完成后,APK 文件位于:

```
G:\fei\android\app\build\outputs\apk\debug\app-debug.apk
```

脚本会自动打开文件夹。

### 步骤 3: 安装到手机

#### 选项 A: 通过 USB + ADB (需要 USB 调试)

1. 手机开启 USB 调试
2. 连接手机到电脑
3. 运行: `.\install-apk.ps1`

#### 选项 B: 文件传输 (最简单)

1. 将 APK 文件发送到手机
   - 通过微信/QQ 发给自己
   - 通过 USB 复制到手机
   - 通过云盘上传下载

2. 在手机上点击 APK 文件

3. 允许安装未知来源应用

4. 点击安装

---

## 一键完成 (如果已开启 USB 调试)

```powershell
.\build-and-install.ps1
```

自动完成: 构建 → 同步 → 安装 → 启动

## 常见问题

### Q: 如何开启 USB 调试?

1. 设置 → 关于手机 → 连续点版本号 7 次
2. 设置 → 系统 → 开发者选项 → 开启 USB 调试

### Q: 手机提示禁止安装?

设置 → 应用管理 → 找到文件管理器 → 允许安装未知应用

### Q: 构建失败?

确保已安装 Java JDK 17+

检查: `java -version`
